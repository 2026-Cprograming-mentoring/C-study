## 🧮 산술 연산자

👉 숫자를 계산할 때 사용하는 연산자

| 연산자 | 설명 | 예제 (a = 10, b = 3) | 결과 |
|--------|------|---------------------|------|
| `+` | 덧셈 | `a + b` | `13` |
| `-` | 뺄셈 | `a - b` | `7` |
| `*` | 곱셈 | `a * b` | `30` |
| `/` | 나눗셈 (몫) | `a / b` | `3` |
| `%` | 나머지 | `a % b` | `1` |

```c
#include <stdio.h>

int main() {
    int a = 10, b = 3;
    printf("%d\n", a + b); // 13
}
