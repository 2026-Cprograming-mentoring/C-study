---

## 1. 메모리와 주소

프로그램에서 모든 변수는 메모리 공간에 저장된다.  
각 변수는 고유한 주소(address)를 가진다.

```c
#include <stdio.h>

int main() {
    int a = 10;
    printf("%p\n", &a); // a의 주소 출력
    return 0;
}
```

- a → 값  
- &a → 주소  

---

## 2. 포인터 (Pointer)

포인터는 주소를 저장하는 변수이다.

```c
int *p;
```

- p는 int형 데이터를 가리키는 포인터

---

### 2.1 포인터 사용

```c
int a = 10;
int *p = &a;
```

- p → a의 주소 저장  
- *p → a의 값  

---

### 2.2 역참조 (Dereference)

```c
printf("%d\n", *p);
```

또는 값 변경:

```c
*p = 20;
```

→ a의 값이 변경됨

---

## 3. 포인터 활용 (swap)

```c
void swap(int *a, int *b){
    int temp = *a;
    *a = *b;
    *b = temp;
}
```

호출:

```c
swap(&x, &y);
```

## 4. 배열 (Array)

```c
int arr[3] = {1, 2, 3};
```

배열은 연속된 메모리 공간이다.

---

### 4.1 배열과 주소

```c
printf("%p\n", arr);
printf("%p\n", &arr[0]);
```

→ 결과 동일

---

### 4.2 배열과 포인터 관계

```c
arr[i] == *(arr + i)
```

예:

```c
printf("%d\n", arr[1]);
printf("%d\n", *(arr + 1));
```

---

### 4.3 포인터로 배열 다루기

```c
int *p = arr;

printf("%d\n", p[1]);
printf("%d\n", *(p + 1));
```

## 6. 동적할당 (Dynamic Allocation)

실행 중에 메모리를 할당하는 방법

---

### 6.1 malloc

```c
#include <stdlib.h>

int *p = (int*)malloc(sizeof(int) * 3);
```

- malloc → 메모리 할당  
- 반환값 → 주소  

---

### 6.2 배열처럼 사용

```c
p[0] = 10;
p[1] = 20;
```

---

## 7. free (메모리 해제)

```c
free(p);
```

- 메모리 누수 방지

---

## 8. 전체 예제

```c
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>
#include <stdlib.h>

int main() {
    int n;
    printf("개수 입력: ");
    scanf("%d", &n);

    int *arr = (int*)malloc(sizeof(int) * n);

    for(int i = 0; i < n; i++){
        arr[i] = i + 1;
    }

    for(int i = 0; i < n; i++){
        printf("%d ", arr[i]);
    }

    free(arr);
    return 0;
}
```

## 9. 핵심 정리

- 포인터: 주소를 저장하는 변수  
- 배열: 연속된 메모리 공간  
- 동적할당: 실행 중 메모리 생성  
- free: 메모리 해제  

---

## 10. 한줄 정리

배열은 포인터를 기반으로 동작하며,  
동적할당은 그 포인터를 통해 메모리를 직접 사용하는 방법이다.
