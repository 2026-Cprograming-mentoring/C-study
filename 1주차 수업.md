
# 📌 1. C 언어란?

C 언어는 1970년대에 만들어진 **절차지향 프로그래밍 언어**로,  
현재까지도 운영체제, 임베디드 시스템, 게임 엔진 등 다양한 분야에서 사용됩니다.

## ✔ 특징
- 매우 빠른 실행 속도
- 메모리를 직접 제어할수있다 (malloc, 포인터등)
- C++, Java 등 많은 언어의 기반

👉 프로그래밍의 기초를 배우기에 매우 좋은 언어

## 예제 코드
```c
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>

int main(){
	printf("광운대 튜터링!\n 화이팅!"); //광운대 튜터링 첫수업

	/*
		주석입니다ㅎㅎ
	*/

  return 0;
}
```

# 📌 2. C 프로그램의 기본 구조

## 🔹 기본 코드

~~~c
#include <stdio.h>

int main() {
    printf("Hello, World!\n"); // 광운대 튜터링 화이팅!!
	/*
		주석입니다ㅎㅎ
	*/

    return 0;
}
~~~

## 🔹 코드 설명

| 코드 | 의미 |
|------|------|
| `#include <stdio.h>` | 표준 라이브러리 헤더 |
| `int main()` | 프로그램 시작할때 가장 먼저 시작되는 함수 |
| `{ }` | 코드의 범위를 나타냄 |
| `printf()` | 화면 출력 |
| `return 0;` | 프로그램 종료 |

👉 모든 프로그램은 main 함수에서 시작

# 📌 3. 변수 (Variable)

## 🔹 변수란?

---

## 🔹 자료형

| 자료형 | 설명 | 예시 |
|--------|------|------|
| int | 정수 | 10 |

| double | 더 정밀한 실수 | 3.141592 |
| char | 문자 | 'A' |

---

## 🔹 변수 선언

~~~c

~~~

# 📌 6. 조건문 (if)

## 🔹 기본 구조

~~~c
if (조건) {
    실행문;
}
~~~

---

## 🔹 if - else

~~~c
int num = 10;

if (num % 2 == 0) {
    printf("짝수\n");
} else {
    printf("홀수\n");
}
~~~

---

## 🔹 else if

~~~c
int score = 85;

if (score >= 90) {
    printf("A\n");
} else if (score >= 80) {
    printf("B\n");
} else {
    printf("C\n");
}
~~~

---

# 📌 7. 반복문 (while)

## 🔹 기본 구조

~~~c
while (조건) {
    실행문;
}
~~~

---

## 🔹 예제 1

~~~c
int i = 1;

while (i <= 5) {
    printf("%d\n", i);
    i++;
}
~~~

---

## 🔹 예제 2

~~~c
int i = 1;
int sum = 0;

while (i <= 10) {
    sum += i;
    i++;
}

printf("%d\n", sum);
~~~
