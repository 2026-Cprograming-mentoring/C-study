
# 📌 1. C 언어란?

C 언어는 1970년대에 만들어진 **절차지향 프로그래밍 언어**로,  
현재까지도 운영체제, 임베디드 시스템, 게임 엔진 등 다양한 분야에서 사용됩니다.

## ✔ 특징
- 매우 빠른 실행 속도
- 메모리를 직접 제어할수있다 (malloc, 포인터등)
- C++, Java 등 많은 언어의 기반

👉 프로그래밍의 기초를 배우기에 매우 좋은 언어

## 예제 코드
```c
#define _CRT_SECURE_NO_WARNINGS
#include <stdio.h>

int main(){
	printf("광운대 튜터링!\n 화이팅!"); //광운대 튜터링 첫수업

	/*
		주석입니다ㅎㅎ
	*/

  return 0;
}
```

# 📌 2. C 프로그램의 기본 구조

## 🔹 기본 코드

~~~c
#include <stdio.h>

int main() {
    printf("Hello, World!\n"); // 광운대 튜터링 화이팅!!
	/*
		주석입니다ㅎㅎ
	*/

    return 0;
}
~~~

## 🔹 코드 설명

| 코드 | 의미 |
|------|------|
| `#include <stdio.h>` | 표준 라이브러리 헤더 |
| `int main()` | 프로그램 시작할때 가장 먼저 시작되는 함수 |
| `{ }` | 코드의 범위를 나타냄 |
| `printf()` | 화면 출력 |
| `return 0;` | 프로그램 종료 |
| `//` | 한줄 주석 |
| `/**/` | /**/은 여러줄 주석 |

👉 모든 프로그램은 main 함수에서 시작

# 📌 5. 연산자

## 산술 연산자

| **연산자**   | **설명**              | **예제 (a = 10, b = 3)**        | **결과**             |
|--------------|-----------------------|---------------------------------|----------------------|
| `+`          | 덧셈                  | `a + b`                         | `13`                 |
| `-`          | 뺄셈                  | `a - b`                         | `7`                  |
| `*`          | 곱셈                  | `a * b`                         | `30`                 |
| `/`          | 나눗셈 (몫)           | `a / b`                         | `3` (정수 나눗셈)     |
| `%`          | 나머지                | `a % b`                         | `1`                  |


## 대입 연산자

| **연산자**    | **설명**              | **예제 (a = 10)**        | **결과 (a의 값)**           |
|---------------|-----------------------|---------------------------------|----------------------|
| `=`           | 대입                  | `a = 5`                         | `5`                  |
| `+=`          | 더하고 대입           | `a += 3`                        | `13`                  |
| `-=`          | 빼고 대입             | `a -= 3`                        | `7`                   |
| `*=`          | 곱하고 대입           | `a *= 2`                        | `20`                  |
| `/=`          | 나누고 대입           | `a /= 2`                        | `5`                   |
| `%=`          | 나머지 대입           | `a %= 3`                        | `1`                   |

## 관계 연산자

| **연산자**    | **설명**              | **예제 (a = 10, b = 3)**        | **결과 (a의 값)**    |
|---------------|-----------------------|---------------------------------|----------------------|
| `==`          | 같음                  | `a == b`                        | `0`                  |
| `!=`          | 다름                  | `a != b`                        | `1`                  |
| `>`           | 큼                    | `a > b`                         | `1`                  |
| `<`           | 작음                  | `a < b`                         | `0`                  |
| `>=`          | 크거나 같음            | `a >= b`                        | `1`                  |
| `<=`          | 작거나 같음            | `a <= b`                        | `0`                  |

## 증감 연산자

| **연산자**    | **설명**              | **예제 (a = 10)**        | **결과 (a의 값)**            |
|---------------|-----------------------|---------------------------------|-----------------------|
| `++a`         | 전위 증가              | `++a`                           | `6`(즉시 증가)        |
| `a++`         | 후위 증가              | `a++`                           | `5`(나중에 증가)      |
| `--a`         | 전위 감소              | `--a`                           | `4`(즉시 감소)        |
| `a--`         | 후위 감소              | `a--`                           | `5`(나중에 감소)      |
```c
int num = 5;
printf("%d", num++);
```
```c
int num = 5;
printf("%d", --num);
```

## 논리 연산자
| **연산자**    | **설명**              | **예제 (a = 1, b = 0)**         | **결과 (a의 값)**     |
|---------------|-----------------------|---------------------------------|-----------------------|
| `&&`          | and                   | `a && b`                        | `0`                   |
| `\|\|`       | or                     | `a \|\| b`                      | `1`                   |
| `!`           | not                   | `!a`                            | `0`                   |

## 비트 연산자
숫자를 2진법으로 바꾸고 각 자릿수별로 논리연산을 한다고 생각하면 편함.
| **연산자**   | **설명**              | **예제 (a = 5, b = 3)**        | **결과**              |
|--------------|-----------------------|---------------------------------|----------------------|
| `&`          | and                   | `a & b`                         | `1`                  |
| `\|`         | or                    | `a \| b`                        | `7`                  |
| `^`          | xor                   | `a ^ b`                         | `6`                  |
| `~`          | not                   | `~a`                            | 보수                 |
| `<<`         | n번 왼쪽으로 시프트    | `a << 1`                        | `10`                 |
| `>>`         | n번 오른쪽으로 시프트  | `a >> 1`                        | `2`                  |

# 📌 6. 조건문 (if)

### if문
```c
if(조건문){
	조건문을 만족할때 실행할 코드;
}
```
### else if문
```c
if(조건문){
	if 조건문이 참일 때 실행할 코드;
}
else if(조건문){
	자신 위의 조건문들이 모두 거짓이며, else if 조건문이 참일 때 실행할 코드;
}
else if(조건문){
	자신 위의 조건문들이 모두 거짓이며, else if 조건문이 참일 때 실행할 코드;
}
else{
	if 조건문과 else if 조건문 모두 거짓일 경우 실행할 코드;
}
```
단 if를 else if 혹은 else와 같이 썼을 때 if ,else if 조건문에서 참이라면 그 뒤에 조건문을 확인하지 않고 다음 코드로 넘어감.

# 📌 7. 반복문 (while)

## 🔹 기본 구조

~~~c
while (조건) {
    실행문;
}
~~~

---

## 🔹 예제 1

~~~c
int i = 1;

while (i <= 5) {
    printf("%d\n", i);
    i++;
}
~~~

---

## 🔹 예제 2

~~~c
int i = 1;
int sum = 0;

while (i <= 10) {
    sum += i;
    i++;
}

printf("%d\n", sum);
~~~
